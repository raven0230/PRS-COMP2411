Valid values for various variables:

Submission.type: 1 digit int
1 - Abstract
2 - Paper
3 - Revised Paper

Submission.reviewStatus: 1 digit int
0 - Pending
1 - Accepted
2 - Rejected

Paper.status: 2 digit int
value = Submission.type * 10 + Submission.reviewStatus
*** It is used to avoid touching Submission table every time when we get paper info.
*** Must be updated every time when there's a new submission / when all reviews are completed.
*** status generator / validator can be found in util\Util class

0: Female
